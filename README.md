# рҹ“– Overview

**StudyBuddy** is an AI-powered peer-to-peer learning platform designed exclusively for college students.

Unlike traditional learning platforms where students learn from professional instructors, StudyBuddy enables students to learn directly from fellow students who recently mastered the same concepts.

The platform helps:

- рҹҺ“ Learners find suitable mentors
- рҹ‘ЁвҖҚрҹҸ« Mentors build reputation
- рҹӨ– AI guide every learning journey

# вқ“ Problem Statement

Many students struggle to:

- Find the right mentor.
- Know where to begin learning.
- Ask questions without fear of judgment.
- Follow a structured roadmap.
- Gain practical skills through projects.

Meanwhile, senior students possess valuable experience but lack a platform to showcase their mentoring abilities.

---

# рҹ’Ў Solution

StudyBuddy connects learners and mentors using Artificial Intelligence.

Instead of randomly searching for mentors, AI analyzes the learner's profile and recommends the most suitable mentor based on:

- Career Goal
- Skill Gap
- Learning Style
- Mentor Expertise
- Availability
- Teaching Style
- Reputation

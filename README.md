# 📖 Overview

**StudyBuddy** is an AI-powered peer-to-peer learning platform designed exclusively for college students.

Unlike traditional learning platforms where students learn from professional instructors, StudyBuddy enables students to learn directly from fellow students who recently mastered the same concepts.

The platform helps:

- 🎓 Learners find suitable mentors
- 👨‍🏫 Mentors build reputation
- 🤖 AI guide every learning journey
- 🪙 Create a skill-credit economy
- ⭐ Build verified mentoring portfolios

---

# ❓ Problem Statement

Many students struggle to:

- Find the right mentor.
- Know where to begin learning.
- Ask questions without fear of judgment.
- Follow a structured roadmap.
- Gain practical skills through projects.

Meanwhile, senior students possess valuable experience but lack a platform to showcase their mentoring abilities.

---

# 💡 Solution

StudyBuddy connects learners and mentors using Artificial Intelligence.

Instead of randomly searching for mentors, AI analyzes the learner's profile and recommends the most suitable mentor based on:

- Career Goal
- Skill Gap
- Learning Style
- Mentor Expertise
- Availability
- Teaching Style
- Reputation

# 🧠 AI Concept

StudyBuddy uses AI to:

- Understand learner goals
- Analyze current skills
- Detect skill gaps
- Recommend mentors
- Suggest projects
- Generate learning paths
- Help mentors prepare sessions

---

# 🪙 Skill Economy

Every learner receives

✅ 100 Credits

Credits are used to:

- Book mentoring sessions
- Unlock advanced mentoring

Mentors earn credits by:

- Teaching students
- Completing mentoring sessions
- Receiving positive reviews

---

# ⭐ Reputation System

Mentors build reputation through:

- Successful sessions
- Student feedback
- Completed learner projects
- Teaching consistency
